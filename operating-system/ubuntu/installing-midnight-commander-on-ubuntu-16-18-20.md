[&larr;](readme.md "Ubuntu") Установка Midnight Commander на Ubuntu 16/18/20
----------------------------------------------------------------------------

Midnight Commander - один из файловых менеджеров с текстовым интерфейсом типа Norton Commander для UNIX-подобных операционных систем.

